# npm库配置

# 临时指定

例如，要下载 node-sass，可以按照下面的命令进行

npm --registry http://nexus.risun-tec.cn:9100/repository/npm-risun i node-sass

# 永久配置

修改 $HOME/.npmrc（没有就创建一个），设置如下内容

registry=http://nexus.risun-tec.cn:9100/repository/npm-risun
