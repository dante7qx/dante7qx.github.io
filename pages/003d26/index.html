科技项目申报流程及注意事项

科技创新是提高社会生产力和综合国力的战略支撑，科技投入是科技创新的物质基础。随着国家对科技创新的资金投入逐年增加，企业可申请的科技项目也越来越多。作为科技创新和成果转化的主体，企业进行科技项目申报可以获得国家及各级政府的财政资金支持，既是支持企业进行科技创新工作的资金来源，也是衡量企业科技实力的指标，可更好的促进企业的科技进步工作。因此，越来越多的企、事业单位积极开展科研项目立项，参与科研项目申报。下面我结合自己的工作经验谈谈作为关键环节的企业科技项目申报工作的工作流程及注意事项。

# 1、项目申报前的信息收集工作

项目申报前准备工作充分，是做好申报工作的前提。

首先，作为企业的科技项目管理人员，应定期在国家、省、市、区级政府部门网站上的通知、公告栏目，查看项目申报或资金支持项目的信息，以免遗漏或者错过申报时机。要及时查阅申报指南的项目支持领域，结合企业所在行业、企业规模等条件分析企业是否符合申报条件。

其次，结合往年的申报及完成情况，对符合申报条件的项目进行分析和筛选，比照申报指南中具体支持项下的要求，确定要申报的项目，使之与项目指南密切贴合。

同时做好项目申报的时间安排和项目申请，如发现有不确定的事项需及时与项目主管部门及时沟通，避免出现理解上的偏差。

再次，对每年各部门的项目申报时间进行规律总结，可以帮助企业科技管理人员提前进行项目规划(如列出未来可预见的2~3年内企业可筹划申报的项目汇总表)，做好项目申报工作。

# 2、申请资料的撰写

申报资料是审评专家评判企业申报项目是否符合要求的唯一的资料，因此申报资料的质量成为项目申请过程中至关重要的环节。一般来讲，项目申报的内容主要包括申请资金的政策依据、该项目的必要性或意义、项目的研究背景和国内外发展趋势、项目的主要研究内容、创新点和拟解决的关键技术问题、现有的工作基础和优势、预期的经济效益和社会效益等内容。

# 2.1 寻求政策支持的有力证据，增强说服力

申请项目需要了解项目的研究领域和行业，是否符合国家、行业、省市相关行业或产业的发展纲要或产业政策的要求，突出与项目密切相关的主题，根据需求进行分层次的阐述。

# 2.2 开门见山突出项目的必要性或意义

项目的必要性或意义需要我们对该项目所在领域有深刻的了解，突出项目目标的重要性。根据项目的实际情况和特点，归纳、总结、有条理的阐述实施该项目的意义和作用，一般从国家宏观发展战略角度、产业链的延伸、社会经济效益、对行业发展的促进作用等方面进行撰写。

# 2.3 梳理项目的研究背景和国内外发展趋势

申报前需对所报项目的发展历程进行横向和(或)纵向梳理，对其发展历程中的重要发展节点进行简要阐述。可归纳为国内外的研究动态，包括每一个研究阶段的特点以及不足之处，现阶段我们的研究进展的程度，总结与以前或者别人比较我们的优势，提出待解决或者已正在解决的问题。

# 2.4 明确项目的研究目标和研究内容

项目的研究目标是项目解决问题以后所要达到的目的。根据企业自身实际情况，我们要把该项目完成后达到的目的和期望值传递给评审专家。一般来讲，包括技术目标、经济目标和学术目标。技术目标可以填补国家、省、市或行业领域的技术空白，推动技术发展和引领行业进步。经济目标可以阐述企业的生产能力，销售收入完成情况和对该地区税收的贡献，以及增加就业等方面，经济目标不可盲目贪大，必要时需要财务部门根据所能达到的产能对销售收入、利税情况进行客观估计。学术目标是通过该项目完成以后企业科研人员发表的论文数量、申请的专利数量或者企业获得的生产批件、行业标准修订等，注意对技术专利、产品专利等的描述不能夸大，避免给项目验收增加难度。研究内容是企业围绕项目完成所做的具体工作，包括研究方法、技术路线等内容。

研究方法需结合企业实际情况，利用现有的人员、设备、技术条件，采用科学合理的途径和手段，达到所要完成的研究目的。技术路线是对项目开始到完成进行的整体设计，阐明整个过程中采用的研究方法，完成的研究内容，解决的关键问题以及达到的预期目标，突出技术、途径以及在拟解决问题等的创新点，凸显技术路线的创新性。在整个过程中，注意措辞不要泄露企业的技术秘密

# 2.5 挖掘创新点和拟解决的关键技术问题

项目的创新点和技术难点是项目申请报告的重点，也是评审专家评分的重要依据。在项目中采用新材料，应用新方法新工艺新技术，创造新产品，这样的项目才能体现创新，可以用专利、技术标准等加以佐证，并对项目的科技产品或技术水平与同行类似产品和技术进行效用、性能对比，分析新产品、新技术的优势和可能为企业带来的经济和社会效益。拟解决的关键问题与研究目的的技术目标相呼应，应仔细分析在项目实施过程中可能会出现的潜在技术问题、其对整个科研项目研究中产生的影响以及进行科技攻关拟采用的途径和手段等

# 2.6 突出现有的工作基础和优势

科技项目资金一般不会支持前期未做任何基础工作的企业，因此企业应把已经做的工作尽可能进行详尽分析。现有的工作基础主要有：与该项目有关的前期的研究基础，企业提供的支撑条件包括设备、仪器、软件以及企业的技术平台等，已承担、完成的科技项目情况和企业所取得的科技成果，项目负责人资历与学术水平，介绍项目团队的人员专业，尤其是职称情况。对与该项目有关的研究工作积累和已取得的成绩应在申请书中进行详实阐述，着重突出企业在人员、技术、资金配套方面的综合实力。

# 2.7 把握预期的经济效益和社会效益

结合企业的规模、产能、销售收入等实际状况，把挥项目实施后带来的经济效益和社会效益。如经济效益一般指为企业带来的直接经济收入，企业的生产能力扩大情况，利税缴纳情况，在市场竞争中使价格降低情况等。社会效益一般可理解为项目实施后为行业、产业以及当地百姓带来的间接效益，如在促进技术进步行业发展、产业链延伸方面、增加就业提高人民生活水平等方面所起的作用。为了保证在后期验收过程中避免申报材料与验收时财务数据不一致，在申报过程中由财务人员根据公司实际情况进行估计，避免出现前后出入太大的现象。

# 2.8 精准取舍相关证明材料

申报资料，要根据项目主管部门的要求附上企业相关的资质证书，如营业执照、高新技术企业证书、企业技术中心证书(各级各类的工程技术研究中心、重点实验室、工程示范研究中心等平台)、资金证明、银行资信等级以及与项目相关的证明材料如科技成果评价/鉴定证书、查新报告、第三方检测报告、专利证书、标准、合同、奖励荣誉等。

# 3、资料的申报

项目申报材料撰写完毕后，需要对各项材料进行整理、检查，确保材料准确无误，注重材料的排版和装订方式，按照要求在规定的时间之前进行网上申报或者送达项目主管部门。

# 结语

以上仅对项目申报过程中的申报材料撰写流程进行的梳理，并对各个环节进行剖析以帮助企业的科技管理人员提升项目申请书的撰写质量，提高项目申请的命中率。项目申报成功靠短期的努力很难取得成功，需要天长日久的积累方能取得。对于科技工作起步晚的企业，即使项目资金扶持力度小甚至无资金扶持，层次低，也要积极申请，作为企业工作基础的积累。另外可与高校、科研院所进行产学研合作，增强申请企业的优势。从企业科研管理的角度来说，整个过程包括信息收集、项目申请、项目实施、项目结题验收等，其中还包括资料撰写、针对项目中涉及的技术问题与科研人员沟通、与主管部门沟通以把握政策信息变化以及事后资料存档等繁琐的工作内容，项目申请只是其中的一部分内容，因此我们作为企业的科技管理人员，在工作中要有全局观念，更好地为企业的科技进步服务。
