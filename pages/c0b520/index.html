ElementUI修改样式

# 一. el-card

修改el-card中header或body样式或自定义图片样式， 参考：https://blog.csdn.net/qq_50276105/article/details/127315880

  1. 修改header样式
.el-card ::v-deep .el-card__header {
  background-color: lightblue;
}
  1. 修改body样式
.el-card ::v-deep .el-card__body {
  background-color: lightblue;
}
  1. 修改el-card中自定义图片样式

  • 默认的样式是：padding：40px,0px

  • 修改样式代码

.el-card ::v-deep .empty {
  padding: 0px;
}
