# Nuget库配置

# 一. 配置

修改 $HOME/.nuget/NuGet/NuGet.Config，设置如下内容

<packageSources>
    <!--
    <add key="nuget.org" value="https://api.nuget.org/v3/index.json" protocolVersion="3" />
    -->
    <add key="nuget-risun" value="http://doc.risun-tec.cn:9100/repository/nuget-risun/index.json" protocolVersion="3" />
</packageSources>
